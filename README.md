# ReplyWise

Inline AI comment generator for LinkedIn built with React, TypeScript, and TailwindCSS.

## Tech Stack
- React 19
- TypeScript
- TailwindCSS 4
- Vite
- CRXJS (Manifest V3)
- OpenRouter API

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built extension will be in the `dist/` folder.

## Load in Chrome

1. Go to `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `dist/` folder

## Features
- Inline AI button in LinkedIn comment editor
- Tone and length customization
- OpenRouter API integration
- Dark mode Vercel-style UI
